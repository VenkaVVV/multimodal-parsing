# 功能介绍

# 适用场景

# 1. 财务要求

将大额资本性支出 (CapEX) 转化为日常经营性支出 (OpEX) 开展财务分析， 避免成本出现大幅波动， 并严格按照权责发生制原则完成账务处理。 当出现集中采购时， 避免出现成本尖刺。

2. 经营分析

支持按月/按天维度对每日成本进行精细化拆解与分析， 更贴合内部成本趋势、 成本结构分布的监控需求， 同时可作为各业务线成本考核与管理的依据。
